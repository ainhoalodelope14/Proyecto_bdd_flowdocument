# Proyectos de FlowDocument y Bases de Datos

Este repositorio contiene dos proyectos de ejemplo en C#:

## Proyecto A – FlowDocument
- Carpeta: `Proyecto2ev_A_AinhoaLL`
- Contiene ejemplos de **FlowDocument** en aplicaciones WPF.
- Permite ver cómo renderizar documentos de manera dinámica en la interfaz.

## Proyecto B – Bases de Datos
- Carpeta: `Proyecto2ev_B_AinhoaLL`
- Contiene ejemplos de **trabajo con bases de datos** (SQL, scripts y conexiones).
- Permite practicar operaciones básicas de creación, lectura y modificación de datos.

## Cómo usar
1. Abrir el proyecto correspondiente en Visual Studio.
2. Para Proyecto A: ejecutar la aplicación WPF para ver los FlowDocuments.
3. Para Proyecto B: ejecutar los scripts en tu motor de base de datos y abrir el proyecto en Visual Studio para probar las conexiones.
